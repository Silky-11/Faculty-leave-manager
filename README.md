# Faculty-leave-manager
IMS Faculty Leave Management System

##  Project Overview
The *JIMS Faculty Leave Management System* is a web-based application designed to simplify and automate the process of applying, approving, and managing faculty leave requests at *JIMS (JaganInstitute of Management Studies)*.

This system reduces paperwork, improves transparency, and ensures efficient leave tracking for both faculty members and administrators.

# Project link

##  Objectives
- To digitize the faculty leave application process  
- To reduce manual work and paperwork  
- To provide quick approval and tracking of leave requests  
- To maintain proper leave records securely

##  How It Works
1. Faculty logs into the system  
2. Leave request is submitted  
3. Admin reviews the request  
4. Request is approved or rejected  
5. Leave data is stored in the database
